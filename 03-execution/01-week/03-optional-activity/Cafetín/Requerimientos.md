# Requerimientos

## Requerimientos Funcionales(RF)

RF1 - Ver productos: Mostrar el catálogo disponible en el cafetín.
RF2 - Agregar al carrito: Permite  añadir productos previamente seleccionados.
RF3 - Modificar cantidad: Permite cambiar la cantidad de productos seleccionada.
RF4 - Calcular total: Permite el cálculo automático del valor total del pedido.
RF5 - Ver pedido: Mostrar pedidos realizados.

## Requerimientos No Funcionales (RFN)

RNF1 - Fácil uso: Interfaz sencilla e intuitiva.
RNF2 - Datos seguros: Protección básica de información.
RNF3 - Respuesta rápida: Procesos menores a 3 segundos.

## Reglas del negocio

RN1 - Disponibilidad de productos: Solo se mostraran los productos disponibles para la venta.
RN2 - Cantidad Mínima: La cantidad mínima permitída por producto es 1 unidad.
RN3 - Confirmación obligatoria: Un pedido solo se registrará cuando el cliente confirme la compra.
