# Alcance del proyecto

## Incluye

1. Visualización del catálogo con los productos disponibles: Permite al usuario consultar los productos ofrecidos por el cafetín con su información básica.

2. Almacenamiento básico de la información: Guarda los datos esenciales de cada pedido en la base de datos.

3. Confirmación del pedido por parte del cliente: Permite finalizar y registrar oficialmente el pedido realizado.

4. Funciones para agregar, modificar o eliminar productos del carrito: Permite gestionar los productos seleccionados antes de confirmar la compra.

5. Cálculo del valor total del pedido automático: El sistema realiza la suma total según los productos y cantidades seleccionadas.

## No incluye

1. Desarrollo de aplicacón móvil nativa: No se desarrollará una app específica para Android o iOS.

2. Implementación de notificaciones automáticas (SMS o correo): No se enviarán mensajes automáticos al cliente cuando el pedido esté listo.

3. Sistema offline: El sistema requerirá conexión a internet para funcionar.

4. Soporte multilenguaje: La aplicación estará disponible únicamente en español.

5. Auditoría avanzada: El sistema no llevará un historial detallado de movimientos o modificaciones, solo guardará la información principal del pedido.
